# 🏢 Rental Management Portal

## 📌 Overview

This project is a full-stack Rental Management Portal designed to manage residential properties, rent payments, and maintenance requests.

The system enables residents to manage their lease details, pay rent, and raise maintenance requests, while allowing admin users to monitor and manage operations efficiently.

It simulates a real-world property management system with role-based access and business workflows.

---

## 🧠 Key Features

### 👤 Resident Features

- Register and login using credentials  
- View lease details (rent, lease duration, documents)  
- Pay rent via bank or card (mock payment system)  
- Submit maintenance requests with severity and category  
- Track request status (Open, In Progress, Closed)  
- Receive notifications for upcoming rent due dates

---

### 🛠️ Admin Features

- View all maintenance requests  
- Update request status and add comments  
- Monitor rent payments and dues  
- Generate rent reports (paid, pending, due dates)
